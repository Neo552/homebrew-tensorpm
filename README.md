# Homebrew Tap for TensorPM

This is the official [Homebrew](https://brew.sh/) tap for [TensorPM](https://tensorpm.com) - AI-powered project management with local-first architecture.

## Installation

```bash
brew tap neo552/tensorpm
brew install --cask tensorpm
```

Or install directly:

```bash
brew install --cask neo552/tensorpm/tensorpm
```

## Updating

```bash
brew upgrade --cask tensorpm
```

## Uninstalling

```bash
brew uninstall --cask tensorpm
```

## What is TensorPM?

TensorPM is a desktop application for AI-powered project management that revolutionizes how you manage projects by embedding an AI that develops a deep, evolving understanding of your entire project context.

**Features:**
- Local-first architecture - your data stays on your device
- Multi-provider AI support (OpenAI, Anthropic, Google, Mistral, Local)
- Cloud sync for team collaboration (optional)
- Cross-platform (macOS, Windows, Linux)

Learn more at [tensorpm.com](https://tensorpm.com)
